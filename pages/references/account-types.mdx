## Account Types

Minimal reference examples for Anchor [account types](https://docs.rs/anchor-lang/latest/anchor_lang/accounts/index.html).

### `Account`

Account container that checks ownership on deserialization.

```rust filename="Type"
Account<'info, T>
```

Examples:
[Playground](https://beta.solpg.io/https://github.com/solana-developers/anchor-examples/tree/main/account-types/Account)
[Github](https://github.com/solana-developers/anchor-examples/tree/main/account-types/Account)

### `AccountInfo`

AccountInfo can be used as a type but Unchecked Account should be used instead.

```rust filename="Type"
AccountInfo<'info>
```

Examples:
[Playground](https://beta.solpg.io/https://github.com/solana-developers/anchor-examples/tree/main/account-types/AccountInfo)
[Github](https://github.com/solana-developers/anchor-examples/tree/main/account-types/AccountInfo)

### `AccountLoader`

```rust filename="Type"
AccountLoader<'info, T>
```

Examples:
[Playground](https://beta.solpg.io/https://github.com/solana-developers/anchor-examples/tree/main/account-types/AccountLoader)
